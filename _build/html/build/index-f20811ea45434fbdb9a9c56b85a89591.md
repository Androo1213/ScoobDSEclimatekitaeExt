title: Getting Started

For first time its best to:

1. **[Installation](installation.md)** — Get Doccker and VS Code set up, open the container
2. **Tutorial notebook** — Open `notebooks/python/Tutorial_Coiled_Setup.ipynb` and work through it! This notebook goes thru Coiled setup, basic library functions, and your first data fetch!
3. **ParkCatalog Demo** — Learn how to search for parks and check what data exists for more data exploratory options.
4. **Spatial Comparison** — Make 2x2 heatmaps comparing climate scenarios across park boundaries

There is no order for notebooks in terms of breaking, I just think its helpful to probably start at the tutorial. You can jump to any one after finishing the installation.
