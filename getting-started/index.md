title: Getting Started
---

If you're new here, this is the order we'd recommend:

1. **[Installation](installation.md)** — Get Docker and VS Code set up, open the container
2. **Tutorial notebook** — Open `notebooks/python/Tutorial_Coiled_Setup.ipynb` and run through it. It walks you through Coiled setup, the library, and your first data fetch.
3. **ParkCatalog Demo** — Learn how to search for parks and check what data exists
4. **Spatial Comparison** — Make 2x2 heatmaps comparing climate scenarios across park boundaries

Each notebook is self-contained — you can jump to any one after finishing the installation.
